# Hammer CRM
Hammer CRM CE bassed in Laravel 5.5.x

## Setup
* Clone: git clone git@github.com:Roskus/hammercrm.git
* Run: ```composer update```
* Check permisions
* Run ```cp .env.example .env```
* Edit .env set language, database.
* Run ```php artisan key:generate```
* Run ```php artisan migrate```
